## 📌 Project Overview
This business case explores operational challenges inside Shopee Express warehouse and identifies opportunities to improve warehouse efficiency through process improvement and AI-assisted solutions.
The objective is to identify warehouse bottlenecks, analyze root causes, evaluate current operational processes, and propose practical solutions that improve productivity without replacing frontline workers.
---

## 💼 Business Problem
### Key Findings
- Manual unpacking increases parcel loss risk
- Parcel searching takes excessive time
- Manual sorting introduces human error
- Parcels delay delivery
- Parcels stuck inside warehouse
- Warehouse operations rely on manual labor

## 🔍 Root Cause Analysis
Based on warehouse observation and process analysis, several root causes were identified:

- Heavy reliance on manual handling
- Manual sorting causing loss and stuck parcels
- No real-time parcel monitoring
- No automatic misplaced parcel detection
- Long searching process
- Limited warehouse visibility
